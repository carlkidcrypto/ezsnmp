=========
CML Tests
=========

Overview
========
This directory contains CML (Command Line Interface) tests for EzSnmp.
These tests are designed to validate SNMP functionality in various scenarios.

Purpose
=======
The CML tests help ensure that EzSnmp works correctly across different:

* Network configurations
* SNMP protocol versions (v1, v2c, v3)
* Concurrent access patterns
* Edge cases and error conditions

Usage
=====
For information on running these tests, please refer to the main development guide at
`Development Documentation <../sphinx_docs_build/source/development.rst>`_.
